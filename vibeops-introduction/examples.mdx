# üéÆ [Labyrinth Puzzle Game](https://dd93fc57-bb74-4477-90f9-b4ba4aec1bd.vercel.app/)

Interactive puzzle game with a cute character navigating through a grid using road cards.

**Sample Prompt:**
```
I want to build a labyrinth game with a cute main character starting from the left, 
trying to reach the right through an unknown grid area. Users get cards representing 
road pieces (crossroads, three-way junctions, straight roads) that they place on the 
grid to help the character move. Only connected roads work, and users can save their 
progress with a username.
```

**Key Features:** Game logic, grid systems, user authentication, animations
